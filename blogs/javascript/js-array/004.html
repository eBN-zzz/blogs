004 JS判断数组中是否包含某个值

Lhotse 2019-11-23 JS数组

# 方法一：array.indexOf

此方法判断数组中是否存在某个值，如果存在，则返回数组元素的下标，否则返回-1。

var arr=[1,2,3,4];
var index=arr.indexOf(3);
console.log(index);

# 方法二：array.includes(searcElement[,fromIndex])

此方法判断数组中是否存在某个值，如果存在返回true，否则返回false

var arr=[1,2,3,4];
if(arr.includes(3))
    console.log("存在");
else
    console.log("不存在");
